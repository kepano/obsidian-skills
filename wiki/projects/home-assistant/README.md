## What It Is

This project covers Tim's personal Home Assistant setup — a self-hosted home automation platform running on a Raspberry Pi. The wiki space collects configuration notes, automation ideas, custom integration work, and hardware setup documentation so that the setup can be reproduced or extended without relying on memory.

## Scope

- Raspberry Pi hardware setup and OS configuration
- Home Assistant installation, integrations, and dashboards
- Custom automations and scripts
- Any custom components or extensions developed for personal use

## Tech Stack

- **Hardware:** Raspberry Pi
- **Platform:** Home Assistant (HASS)
- **Languages:** YAML (automations / config), Python (custom components)
