# MIDI Looper

> [!abstract] Summary
> A Raspberry Pi–based MIDI looper with a touchscreen interface built in Python and Kivy. Designed as a physical music performance tool for recording, layering, and playing back MIDI loops.

---

## What It Is

The MIDI Looper is a hardware/software project combining a Raspberry Pi with a touchscreen display and MIDI I/O. The Kivy framework provides the touch UI, while Python handles MIDI signal processing and loop management. The goal is a standalone, stage-ready looper that doesn't require a laptop.

## Tech Stack

- **Hardware:** Raspberry Pi + touchscreen display + MIDI interface
- **Software:** Python / Kivy
- **Domain:** Music, live performance, embedded systems
