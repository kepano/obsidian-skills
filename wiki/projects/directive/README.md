# Directive

> [!abstract] Summary
> Directive is a conversational process-creation hub — described as "GitHub for process models." It enables teams to define, version, and share operational processes through a structured, collaborative interface.

---

## What It Is

Directive is a web application that treats business processes as first-class artefacts: versionable, linkable, and composable. Users can create process models through a conversational interface, browse a shared library of processes, and fork or adapt existing ones — much like a code repository but for workflows and SOPs.

## Tech Stack

- **Frontend:** React / Vite / TypeScript
- **Backend:** Node.js / Express
- **Target users:** Teams and organisations that need to document, share, and iterate on operational processes
