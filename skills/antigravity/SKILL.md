# Antigravity

Antigravity is an advanced agentic coding assistant designed by Google DeepMind. It is built to pair program with users, solve complex coding tasks, and interact with codebases effectively.

## Capabilities

- **Agentic Execution:** Can autonomously navigate codebases, read files, and write code.
- **Tool Use:** Equipped with powerful tools for search, file manipulation, and environment interaction.
- **Context Awareness:** Understands user requests within the context of the active workspace.
