# Gemini CLI

Use the Gemini CLI to interact with Google's Gemini models directly from the terminal. 

## Usage

Basic prompting:

```bash
gemini prompt "Your prompt here"
```no

You can use the CLI to test prompts, analyze local files, or quickly query the Gemini API without leaving the terminal.
